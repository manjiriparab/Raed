# Raed — Bilingual AI Companion

Talk to Raed in English or Arabic. She understands both and replies in the same language you use — with voice output and a 2D animated avatar.

## Tech Stack

- **Next.js 14** — fullstack framework
- **Claude Haiku API** — AI brain (fast + cheap)
- **Web Speech API** — voice input & output (free, browser built-in)
- **Custom SVG Avatar** — 2D animated character with states

## Setup

### 1. Install dependencies
```bash
npm install
```

### 2. Run
```bash
npm run dev
```

Open http://localhost:3000

## Features

- 🎙 Voice input — hold mic button and speak
- ⌨️ Text input — type and press Enter or send button
- 🌍 Bilingual — English and Arabic
- 🔊 Voice output — Raed speaks his replies
- 😊 Animated avatar — reacts to idle / listening / thinking / talking states
- 💬 Conversation memory — remembers the full chat context
